# 🐍 Python Programmiersprache: Eine umfassende Einführung

## 🤔 Was ist Python?

Python ist eine moderne, interpretierte Programmiersprache, die sich durch ihre **Einfachheit** und **Vielseitigkeit** auszeichnet. Sie wurde entwickelt, um sowohl für Anfänger als auch für erfahrene Programmierer zugänglich zu sein.

## 📚 Die Geschichte von Python

### 🧑‍💻 Entstehung und Entwickler
Python wurde **1991** von **Guido van Rossum** entwickelt, einem niederländischen Programmierer, der eine einfache und elegante Programmiersprache schaffen wollte.

### 🎭 Der überraschende Namensgeber
Anders als viele vermuten, ist Python **nicht nach der Schlange benannt**! 

Der Name stammt von der britischen Komiker-Gruppe **"Monty Python"** - ein Hinweis auf den humorvollen Ansatz der Entwickler.

## ⚙️ Technische Eigenschaften

### 🔄 Interpretierte Sprache
- Python ist eine **interpretierte Programmiersprache**
- Code wird zur Laufzeit ausgeführt, nicht vorher kompiliert
- **🌐 Plattformübergreifend**: Läuft auf Windows, macOS und Linux
- **🏗️ Objektorientiert**: Unterstützt moderne Programmierparadigmen

### ✨ Die besondere Syntax
Python verwendet eine **einzigartige Syntax-Philosophie**:

- **📐 Einrückungen** strukturieren den Code (statt geschweifte Klammern)
- **📖 Klare, lesbare Syntax** - "Code soll wie Prosa lesbar sein"
- **🎯 Ideal für Programmier-Einsteiger** durch intuitive Struktur

**Beispiel für Python-Syntax:**
```python
def greet_user(name):
    if name:
        print(f"Hallo, {name}!")
    else:
        print("Hallo, unbekannter Benutzer!")

greet_user("Maria")
```
