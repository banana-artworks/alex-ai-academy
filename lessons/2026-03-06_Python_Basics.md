# 🐍 Python Basics: Eine strukturierte Einführung

## 🤔 Was ist Python?

### 🏗️ Entstehung und Philosophie
Python wurde **1991** von Guido van Rossum entwickelt und folgt der **"Zen of Python"** Philosophie mit dem Leitsatz:

> 📖 **"Readability counts"** - Lesbarkeit und einfacher, eleganter Code stehen im Vordergrund.

---

## ⚙️ Wie funktioniert Python?

### 🔄 Interpretierte Sprache
Python ist eine **interpretierte Sprache**, das bedeutet:
- Der Code wird zur Laufzeit **Zeile für Zeile** ausgeführt
- Keine vorherige Kompilierung nötig
- Ermöglicht **interaktives Programmieren** und schnelle Entwicklungszyklen

### 🌐 Plattformunabhängigkeit
Python-Code läuft unverändert auf verschiedenen Betriebssystemen:
- ✅ **Windows**
- ✅ **macOS** 
- ✅ **Linux**

Dies vereinfacht die **Portabilität** von Anwendungen erheblich.

---

## ⭐ Besonderheiten der Python-Syntax

### 🔀 Dynamische Typisierung
Variablen in Python müssen **nicht explizit deklariert** werden - der Datentyp wird automatisch zur Laufzeit bestimmt:

```python
x = 10        # Integer
x = "Hello"   # String (gleiche Variable, neuer Typ)
x = [1, 2, 3] # Liste (erneute Typänderung)
```

### 📏 Einrückungsbasierte Syntax
Python nutzt **Einrückungen** (Indentation) statt Klammern zur Strukturierung:

```python
if x > 5:
    print("Größer als 5")        # 4 Leerzeichen Einrückung
    print("Weitere Anweisung")   # Konsistente Einrückung
else:
    print("Kleiner oder gleich 5")
```

> 💡 **Tipp**: Dies erzwingt automatisch sauberen, lesbaren Code!
